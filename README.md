# Analysis-Plus-AI

تحليل بلس – نظام تحليل بيانات فوري

Analysis Plus هو نظام ويب مبني على Flask، يساعدك على:

استكشاف البيانات وقواعد البيانات بسهولة.

طرح أسئلة طبيعية وتحويلها تلقائيًا إلى استعلامات SQL وتنفيذها.

التفاعل مع نموذج ذكاء اصطناعي عبر OpenRouter للحصول على تحليلات وشرح النتائج.

المتطلبات

Python: إصدار 3.10 أو أحدث

بيئة عمل افتراضية (مستحسنة)

تثبيت المكتبات من requirements.txt

مفتاح OpenRouter API من حسابك على OpenRouter

الإعداد
1. تثبيت الحزم
pip install -r requirements.txt
2. تهيئة ملف البيئة .env

أنشئ أو عدّل ملف .env وضع فيه القيم الأساسية (لا تشارك المفاتيح الحساسة):

# قاعدة بيانات حقيقية (اختياري)
DATABASE_URL=postgresql://user:pass@host:5432/dbname

# قاعدة بيانات تجريبية (Demo)
DEMO_DATABASE_URI=sqlite:///mock.db

# مفتاح OpenRouter API
OPENROUTER_API_KEY=sk-or-xxxxxxxxxxxxxxxx

⚠️ يُفضل عدم رفع ملف .env إلى GitHub أو مشاركته علنًا.

3. تشغيل الخادم
python app.py

عادةً سيفتح الخادم على:
http://localhost:5000

طريقة الاستخدام
الصفحة الرئيسية

تعرض صفحة هبوط للنظام مع خيارات الاستكشاف والتجربة.

وضع اللعب/الاستكشاف

Demo: تجربة بيانات وهمية بأمان.

Real: العمل مباشرة على بيانات حقيقية.

ربط قاعدة بيانات خارجية (PostgreSQL) لقراءة الـ schema وتوليد بيانات وهمية للشرح.

رفع ملفات البيانات

Excel (.xlsx, .xls)

SQLite (.db, .sqlite)
✅ يتم تحويلها تلقائيًا لقاعدة بيانات نشطة وتحضيرها للتحليل.

طرح الأسئلة

مثال: "ما عدد الزيارات خلال آخر أسبوع؟"

النظام يحوّل السؤال إلى SQL، ينفذه، ويعرض النتائج مع اقتراح رسم بياني مناسب.

صفحة الدردشة

عبر /chat يمكن التفاعل مع نموذج الذكاء الاصطناعي للحصول على تحليلات وشرح البيانات بشكل مباشر.

صفحات إضافية

/about و /contact لتقديم معلومات عن النظام والتواصل.

الملاحظات الأمنية

لا تشارك ملف .env أو مفاتيح API في المستودعات العامة.

عند استخدام بيانات حقيقية، احرص على الالتزام بسياسات الخصوصية وعدم رفع قواعد البيانات الحساسة خارج بيئتك الآمنة.
